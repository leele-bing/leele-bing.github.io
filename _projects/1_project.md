This repository hosts my personal website at [leele-bing.github.io](https://leele-bing.github.io).

The site is based on [al-folio](https://github.com/alshedivat/al-folio), configured for a user GitHub Pages repository, and currently includes:

- a personal homepage with profile photo;
- a custom favicon;
- a compact project page;
- a GitHub repositories page connected to the `leele-bing` account.
